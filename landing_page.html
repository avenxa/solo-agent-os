Prompt 01 MLS Listing Description
⬤   BC Context  ·  BCMLS Standards  ·  RESA Accuracy Obligations
The most time-consuming writing task in every listing — done in 2 minutes with a professional, MLS-ready description that doesn't start with "Welcome to this beautiful home."
You are a licensed BC real estate agent writing an MLS listing description...

Property details:
- Address: [full address, city, BC]
- List price: [$XXX,XXX]
- Key features: [list 4–6 features]
- Unique selling point: [the one thing...]

Write a listing description that opens with a compelling hook — not "Welcome to this beautiful home." Highlights the features in natural, conversational language...
BC Note   Verify all measurements before publishing — agents are responsible for accuracy under RESA. No discriminatory language under BC Human Rights Code.
Prompt 02 Price Reduction Conversation Prep
⬤   BC Context  ·  BCFSA Conduct  ·  Fiduciary Duty
The hardest conversation in a listing — prepared with data-backed rationale, empathetic framing, and three objection responses so you walk in confident.
You are a licensed BC real estate agent preparing for a price reduction conversation...

- Days on market: [X days]
- Number of showings: [X]
- Comparable sales: [1–3 comps with prices]
- Proposed new price: [$XXX,XXX]

Prepare: (1) a brief summary of market signals, (2) a script for how to open the conversation — empathetic, direct, not apologetic, (3) three objection responses...
BC Note   Under BCFSA conduct rules, you have a duty to be honest and forthright with your seller client. Presenting market data is part of your fiduciary obligation.
Prompt 03 Post-Showing Feedback Request
⬤   Seller Advisory  ·  Showing Intelligence
Most agents never follow up after showings. This gives you a professional, non-pushy message to send the same evening — email and a 50-word text version.
You are a licensed BC real estate agent following up after a showing...

- Showing date: [date and time]
- Showing agent: [name, if known]
- Buyer context: [e.g. first-time buyer]

Write two versions: one for email, one for a short text message under 50 words. Warm, collegial, brief. Asks three specific questions about impression, comparison, and concerns...
BC Note   Feedback should be shared with your seller client. Sellers have the right to know how buyers are receiving their property.
